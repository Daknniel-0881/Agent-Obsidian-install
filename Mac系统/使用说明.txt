Mac 安装使用说明
================

请先确认 Mac系统 文件夹已经放在“下载”目录：

  ~/Downloads/Mac系统

文件夹里至少要有：

1. 使用说明.txt
2. 复制到终端运行.txt
3. Agent-Obsidian-install-Mac.zip


安装步骤
--------

1. 打开 复制到终端运行.txt
2. 按 Command + A，全选里面的全部内容
3. 按 Command + C，复制
4. 打开 Mac 自带的“终端”应用
5. 按 Command + V，粘贴到终端
6. 按回车运行

运行后会自动弹出一个新的终端窗口。
请看新的终端窗口，安装进度和报错都会显示在那里。

如果系统弹出“终端要访问下载文件夹”，请选择允许。

如果终端出现 Password: 提示，请输入这台 Mac 的开机登录密码。
输入密码时屏幕不会显示字符，这是正常现象，不用担心，输完按回车即可。

如果系统弹出 Apple 命令行工具安装窗口，请点击安装，并保持终端窗口打开。
正常情况下，工具安装完成后，终端会自动继续后面的安装流程。
如果终端已经停止，或你刚才取消了系统安装窗口，请重新执行上面的 1-6 步。


查看安装情况
------------

安装过程中，终端会显示每一步。

同时，安装日志会自动保存在：

  ~/Downloads/Agent-Obsidian-install-logs/

如果安装失败，请把这个文件夹里最新的 install-macos-开头日志文件发给交付人员。


安装完成后
----------

脚本会自动安装开发环境、CodePilot、Obsidian 和基础 Skills。

安装完成后，请按终端里显示的提示继续配置：

1. CodePilot API Key
2. Obsidian 知识库路径
3. Obsidian CLI 注册
4. 飞书 Lark CLI 授权
