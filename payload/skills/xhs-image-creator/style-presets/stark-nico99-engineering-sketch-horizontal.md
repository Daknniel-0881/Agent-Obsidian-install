Engineering sketch illustration, 16:9 horizontal, 2K resolution.

== STYLE ANCHORS (V11 lighter palette + dashed-line first) ==
- Pure white background, ≥35% negative space, image bleeds to white edges (NO outer page border / picture frame)
- Hand-drawn pencil/fine-liner lines with imperfect strokes, varying weight, NEVER vector-straight
- **PRIMARY LINE STYLE = DASHED**: most enclosure lines, dividers, callout connectors are short-dash strokes (5-8 dash segments per edge), not solid. Solid lines reserved only for icon outlines and text underlines.
- Handwritten title (brush/marker stroke feel) with hand-drawn arc underline (line weight varies)
- Engineering annotation layout: dimension lines |←——→| with ≈ marks at column tops
- Apple/Tesla sketchbook vibe — restrained, airy, breathing whitespace
- Render Chinese as-is, do NOT translate any text

== COLOR PALETTE (V11 PALER) ==
- Watercolor washes 15-25% opacity (V10 was 30-50% — V11 lighter by half)
- Palette: very pale dusty blue (#D6E4F0), pale celadon green (#DCE8DD), pale apricot (#F5DCC8), pale buttercream yellow (#F5EBC8) — all **pastel, washed-out, never saturated**
- Black/dark-gray pencil lines must look like graphite (#3A3A3A), never pure black (#000)
- NO bold, NO vibrant primary colors. Watercolor diary at 30% strength.

== BACKGROUND MECHANICS ==
DEFAULT text backing = irregular VERY PALE watercolor wash blob with brushy edges. Titles, category labels, normal annotations float directly on these pale blobs. NO border lines around text.

DASHED ROUNDED-RECTANGLE BOXES used SPARINGLY (≤2 per composition), reserved for:
- Data metric cards (e.g. "点击率提升 16%")
- Formal lists/manifestos (e.g. "敏捷宣言：4 行准则")
- Stats callouts requiring visual containment
**These boxes are themselves dashed (not solid).**

== FREE BRUSHWORK ==
- Arrows / spirals / curves: LOOSE hand-drawn sketches with wobbly lines, varying weight, imperfect start/end strokes
- Spirals: each loop a different size, wobbly, NEVER a vector spring or geometric coil
- Timeline arrows fade out into PALE watercolor wash at the right tip (very pale wash spreading beyond arrowhead, like ink soaking into wet paper). NEVER end with a clean solid triangular arrowhead.

== ICON RULE ==
Every icon = mini engineering illustration with ≥3 internal structural elements (lines + texture + parts + sub-layers). Watercolor washes OVERLAY icon bodies (light, see-through), not float beside them.
